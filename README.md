# DClock 
本插件由DIL工作室制作
用DClock来获取钟吧

DClock 是一个LiteLoaderBDS的插件

本插件可以获取钟表(钟表在有些菜单插件中是菜单)
插件简介：DClock 是一款给钟插件，同时也带有进入提示
新功能：每个玩家只能获取一次钟，除了/clear指令其他均无法清除（BETA）
